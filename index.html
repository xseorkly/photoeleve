📋 À propos de PhotoBadge
PhotoBadge est une application web gratuite pour les établissements scolaires. Elle permet de photographier les élèves, de générer automatiquement leur badge personnalisé, d'enregistrer les photos et les badges directement sur votre ordinateur ou téléphone et d'archiver les photos sur Google Drive si vous le désirez — Comment configurer Google Drive ? — le tout depuis un navigateur web, sans aucune installation.

📸 Capture photo — caméra ou import d'image
🪪 Badge automatique — nom, classe, logo, couleurs de l'école
🖨 PDF A4 — 8 badges par page, prêts à imprimer
☁ Google Drive — archivage automatique des photos (optionnel)
💾 Sauvegarde locale — enregistrement sur votre ordinateur ou téléphone

❓ Guide d'utilisation

1 — Configurer Google Drive

1

Allez sur Google Cloud Console : console.cloud.google.com et créez un projet.

2

Dans APIs & Services → Bibliothèque, cherchez Google Drive API et cliquez Activer.

3

Allez dans Google Auth Platform → Premiers pas. Remplissez le nom de l'application et votre email, puis enregistrez.

4

Dans Audience, ajoutez votre adresse Gmail dans les Utilisateurs test.

5

Dans Clients → Créer un client, choisissez Application Web. Dans Origines JavaScript autorisées, ajoutez l'URL de votre site (ex: https://monsite.github.io).

6

Copiez l'ID client obtenu (format : xxxxx.apps.googleusercontent.com) et collez-le dans les paramètres de l'application.

💡 Conseil : Créez un dossier PhotoDrive dans votre Google Drive. Ouvrez-le et copiez l'ID dans l'URL (après /folders/). Collez cet ID dans le champ "ID dossier Drive" des paramètres.

2 — Paramétrer l'application

1

Cliquez sur ⚙ Paramètres dans le header pour accéder à la configuration.

2

Renseignez le nom de l'établissement, la ville et l'année scolaire.

3

Ajoutez le logo via un lien URL ou en uploadant directement le fichier image (JPG, PNG).

4

Choisissez les couleurs principale et secondaire pour personnaliser les badges.

5

Collez votre Client ID Google OAuth et l'ID du dossier Drive si vous utilisez Google Drive.

3 — Utiliser l'application

1

Activez la caméra en cliquant sur la zone vidéo, ou importez une image avec le bouton 📁.

2

Remplissez la classe, le nom et le prénom de l'élève. Le badge se génère en temps réel.

3

Cliquez ☁ Envoyer sur Drive pour sauvegarder la photo sur Google Drive (nécessite une connexion Google).

4

Cliquez 🪪 Générer le badge pour ajouter le badge à la liste. La photo est aussi sauvegardée localement au format JJ-MM-AAAA-HH-MM-SSs.jpg.

5

Une fois tous les badges générés, cliquez 👁 Visualiser PDF pour prévisualiser ou 🖨 Télécharger PDF pour sauvegarder. Max 8 badges par page A4.

4 — Sauvegarde locale

1

Chaque photo prise est automatiquement téléchargée : 20-03-2026-23-00-14s.jpg.

2

Les PDF téléchargés suivent le même format : badges_20-03-2026-23-00-14s.pdf.

3

Sur Chrome/Edge, un sélecteur de dossier s'ouvre pour choisir où sauvegarder.

⚠️ Firefox : Le choix de dossier n'est pas supporté. Les fichiers iront dans le dossier par défaut.
